Welcome to LoL Item Set Editor

Hello newcomer! This tool is made as a replacement for the Item Set Editor in the new client for League of Legends.

I always liked using customized sets, so since the feature is gone for who-knows-how-long in the new client, here we are. If you were used to managing your Sets, you should find your marks easily. A few important things though:
  • This tool is web based only, it cannot read or deploy your sets itself.
  • Item sets are stored within the League of Legends config directory. Usually it is located in C:\Program Files(x86)\Riot Games\League of Legends\Config\
  • To load your custom Sets into the tool, drop your folder on this window.
  • To apply your Sets, click the download button and replace the Config folder with the content of the .zip archive.
  • This tool will auto update itself every few hours, so you should see new patch items rather quickly. Current patch: {{version}}
Hope this can be of use to you, and best luck on the Rift! (unless you face me of course)
