Changelog

03/05/2017 - Version 0.15

  • Fix downloading on Vivaldi browser
  • Hard limit of 2000 files removed. Instead will display a confirmation message when trying to upload more than 2000 files.
  • Detecting "type:'global'" for some edge case of global sets stored by Riot in champions folders.

20/04/2017 - Version 0.14

  • Display gold total for each block
  • You can now get statistics for a block by clicking on the  button on the block. Be aware that these stats are incomplete, since the Riot API does not give structured stats for some attributes (MP5 and HP5 for example)

19/04/2017 - Version 0.13

  • You can now import builds from LoLalytics

05/04/2017 - Version 0.12

  • Fix folder structure when downloading a .zip file

01/04/2017 - Version 0.11

  • Scrollable when list of shared champions too long
  • Download item-set on Firefox works again
  • Better display of popover for items in blocks

15/02/2017 - Version 0.10

  • You can now create Sets used by multiple Champions (Use "Add another champion" with the button)
  • You can now delete all sets from a Champion
  • Fixed downloading for Opera browser
  • Confirmation boxes are now built-in, to avoid the browser hiding them
  • Fixed creation of Global sets

15/01/2017 - Version 0.9

  • Fixed an issue with Riot default item sets not behaving correctly

07/01/2017 - Version 0.8

  • You can now import builds from Mobafire
  • You can now import builds from LoLKing
  • New layout for the Upload page

05/01/2017 - Version 0.7

  • Adding a new Set now loads the default recommended set for Summoner Rift
  • New menu to import a recommended item set for a champion ( button)
  • Added "builds into" in item popover
  • Added "Vision" & "Jungle" categories
  • Jungle items should now filter correctly on AD/AP/HP/AS/CDR
  • Fixed scrolling issue when adding the first set to the tool

03/01/2017 - Version 0.6

  • Added chart of build path for complex items

30/12/2016 - Version 0.5

  • Added a sharing link to share a set via an URL ( button)

  • Deprecated items will no longer show when loading old sets
  • Sets will now try to keep the filename they were loaded with (when loaded and not created ad hoc)

  • Changed application icon to not use the League of Legends logo
